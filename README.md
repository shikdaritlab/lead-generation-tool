# Legal Lead Generation Tool

A fully compliant B2B lead generation application built with Next.js, using only legal public databases and open data sources.

## ✅ Features

- **Legal Data Sources Only**: Uses verified government registries and official public databases
- **GDPR & CCPA Compliant**: Fully meets international data protection regulations
- **Global Coverage**: 12 Tier-1 countries with state/region filtering
- **CSV Export**: One-click export of all results
- **Real-time Search**: Search 100-10,000+ leads per query
- **Responsive Design**: 1100px centered container, mobile optimized
- **No Sidebar**: Clean, distraction-free interface
- **Free & Open**: Uses only free, legal APIs

## 📊 Data Extracted

- ✅ Company Name

## 📡 Legal Data Sources

1. **OpenCorporates** (https://opencorporates.com) - Global company registry
2. **SEC EDGAR** (https://www.sec.gov/cgi-bin/browse-edgar) - US public company filings
3. **Companies House** (https://www.companieshouse.gov.uk) - UK company registry
4. **ASIC** (https://www.asic.gov.au) - Australian company data
5. **Ministry of Corporate Affairs** (https://www.mca.gov.in) - Indian business registry
6. **Handelsregister** (https://www.handelsregister.de) - German business registry
7. **ACRA Registry** (https://www.acra.gov.sg) - Singapore company registry
8. **OpenStreetMap** (https://www.openstreetmap.org) - Geographic business locations

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/shikdaritlab/lead-generation-tool.git
cd lead-generation-tool
npm install
npm run dev
```

Open http://localhost:3000

## 📦 Build & Deploy to Vercel

```bash
npm run build
git add .
git commit -m "Deploy to Vercel"
git push origin main
```

Then go to https://vercel.com/new and import the repository.
